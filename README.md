# 🌱 Objetivo do Projeto

O descarte irregular de móveis é um problema frequente em áreas urbanas, causando:

* poluição visual
* proliferação de pragas
* ocupação indevida de espaços públicos
* impactos ambientais

O **Móveis Já** foi criado para facilitar o acesso a informações sobre **locais adequados para descarte** e incentivar a população a **denunciar situações de descarte inadequado**, contribuindo para uma cidade mais limpa e sustentável.
