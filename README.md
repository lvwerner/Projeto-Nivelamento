📦 Móveis Já










Sistema web desenvolvido para auxiliar no descarte correto de móveis, permitindo que usuários encontrem pontos de coleta e realizem denúncias de descarte irregular que prejudicam o meio ambiente.

O projeto foi desenvolvido como parte da disciplina Fábrica de Software, com foco na aplicação de conceitos de desenvolvimento web, organização de projeto e prototipação de interfaces.

🌐 Acesso ao Sistema

A aplicação está disponível online:

🔗 https://projeto-nivelamento.vercel.app

O sistema pode ser acessado diretamente pelo navegador, sem necessidade de instalação.

🌱 Objetivo do Projeto

O descarte irregular de móveis é um problema frequente em áreas urbanas, causando:

poluição visual

proliferação de pragas

ocupação indevida de espaços públicos

impactos ambientais

O Móveis Já foi criado para facilitar o acesso a informações sobre locais adequados para descarte e incentivar a população a denunciar situações de descarte inadequado, contribuindo para uma cidade mais limpa e sustentável.
