📦 Móveis Já

Sistema web desenvolvido para auxiliar no descarte correto de móveis, permitindo que usuários encontrem pontos de coleta e realizem denúncias de descarte irregular que prejudicam o meio ambiente.

O projeto foi desenvolvido como parte da disciplina Fábrica de Software, com o objetivo de aplicar conceitos de desenvolvimento web e prototipação com Figma.

🌱 Objetivo do Projeto

O descarte irregular de móveis é um problema comum em áreas urbanas, causando:

poluição visual

proliferação de pragas

ocupação indevida de espaços públicos

impactos ambientais

O Móveis Já busca facilitar o acesso a informações sobre locais adequados para descarte e incentivar a população a denunciar situações de descarte inadequado.

🖥️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias básicas da web:

HTML5

CSS3

JavaScript

Figma (prototipação da interface)

Não foram utilizados frameworks para manter o projeto simples e didático.

📌 Funcionalidades

O sistema possui as seguintes funcionalidades:

🏠 Página Inicial

Apresenta a proposta do projeto com uma imagem representando a importância da preservação ambiental.

♻️ Pontos de Coleta

Lista locais responsáveis pela coleta de móveis descartados.

Cada ponto possui:

organização responsável

link para o site oficial

link para localização no Google Maps

📚 Informações Importantes

Apresenta orientações sobre o descarte correto de móveis e os impactos ambientais do descarte irregular.

🚨 Denúncias

Formulário onde o usuário pode informar:

nome

bairro

telefone

descrição do problema
